# 프로젝트 기본 골격

## 목표

서울 지역의 공공데이터를 한 곳에서 조회하고, 익명 게시글과 AI 챗봇으로 정보를 탐색하는 정적 SPA를 만든다.

## MVP

### Must

- Vue 3 + Vite 정적 SPA
- 선정 권역 JSON 화면 표시 및 검색
- 게시글 목록, 상세, 작성, 수정, 삭제
- localStorage 저장 및 비밀번호 검증
- 챗봇 입력, 응답, 현재 화면 대화 이력
- 관련 지역정보 및 현재 브라우저 게시글만 추려 OpenAI API에 전달
- 데스크톱 플로팅 챗봇, 모바일 전체 화면
- Netlify 배포
- 기능명세서, WBS, 발표자료, 데이터 출처 문서

### Should

- 게시글 제목·내용 검색
- 카테고리 필터
- 삭제 비밀번호 모달
- API 로딩·실패·시간 초과 상태
- 빈 데이터 및 잘못된 경로 처리

### Could

- 지도 시각화
- 축제 캘린더
- 데이터 시각화 대시보드

### Won't

- 백엔드 서버와 서버 DB
- 회원가입과 로그인
- 다른 브라우저 사용자와 게시글 공유
- WebSocket 알림
- 라이선스 미확인 데이터

## 핵심 데이터 구조

```js
{
  id: 'uuid',
  region: 'seoul',
  category: 'free',
  title: '게시글 제목',
  content: '게시글 내용',
  password: '1234',
  createdAt: '2026-07-14T10:30:00.000Z',
  updatedAt: '2026-07-14T10:30:00.000Z'
}
```

## 화면 흐름

```text
홈
├─ 커뮤니티 목록
│  ├─ 게시글 작성
│  └─ 게시글 상세
│     ├─ 수정
│     └─ 삭제 비밀번호 모달
├─ 지도 선택 기능
└─ 축제 일정 선택 기능

모든 화면
└─ 챗봇 플로팅 버튼
   └─ 데스크톱 패널 / 모바일 전체 화면
```

## 기술 결정

- 페이지 컴포넌트는 표시와 사용자 이벤트만 담당한다.
- CRUD 상태 흐름은 `usePosts.js`가 담당한다.
- 저장은 `postStorageService.js`로 제한한다.
- JSON 검색과 필터는 `regionDataService.js`로 제한한다.
- OpenAI 요청, 타임아웃, 응답 파싱은 `openaiService.js`로 제한한다.
- 신규 선택 기능은 Backlog에 먼저 기록하고 필수 기능 완료 후 진행한다.
